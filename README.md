# Hidden-Catch-Project

## Project Synopsis 👀 
<div>

### Description
🧠 다운로드나 설치 없이 간단한 가입 후 WEB 상에서 할 수 있는 숨은그림찾기 게임! <br>
친구들과 누가누가 더 잘하나 승부를 겨뤄보세요!

### 선정이유
- WEB과 DB를 연동한 프로젝트
- CRUD 구현
- HTML/JSP를 활용한 화면간 이동 구현
- Web-Based의 처리는 Request/Response구조라는 웹의 특징을 활용하여 구현됨
